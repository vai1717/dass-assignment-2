\section*{\textcolor{blue}{Introduction}}
This report documents black-box testing of the QuickCart REST API using \texttt{pytest} and \texttt{requests}. All 93 test cases were designed from the API documentation without access to implementation. Tests verify correct HTTP status codes, JSON response structures, and data correctness.

\textbf{Result: 74 passing, 19 bugs found.}

\section*{\textcolor{blue}{Test Cases}}

Each test case lists: Category, Input (Method + URL + Headers + Body), Expected Output, and Justification.
